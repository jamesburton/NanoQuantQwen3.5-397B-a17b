  <action>
Run Stage 1 quantization on OLMoE-1B-7B-0924:

```bash
python scripts/run_stage1.py \
  --model allenai/OLMoE-1B-7B-0924 \
  --output output/OLMoE-1B-7B-0924 \
  --bits-per-weight 0.80 \
  --admm-iters 50 \
  --n-calibration 128 \
  --seq-len 2048 \
  --checkpoint-dir checkpoints/OLMoE-1B-7B-0924
```

After quantization completes, run evaluation:

```bash
python scripts/run_eval.py \
  --model allenai/OLMoE-1B-7B-0924 \
  --quantized-dir output/OLMoE-1B-7B-0924 \
  --results-dir results \
  --stride 512 \
  --max-length 2048
```

Verify the checkpoint is loadable: confirm `output/OLMoE-1B-7B-0924/` contains `quantized_factors.safetensors` and `quantized_manifest.json`.

Verify metrics: confirm `results/OLMoE-1B-7B-0924/metrics.json` contains `perplexity.fp16_baseline`, `perplexity.quantized`, `hardware.peak_vram_gb`, `model_size.fp16_gb`, `model_size.quantized_gb`.

If quantization fails (OOM or error), document the failure in results notes and try with `--max-blocks 2` as a diagnostic dry run first.
