Add an optional `bits_per_weight: Optional[float] = None` parameter to `quantize_model()`. If provided, compute a suggested rank from it using the formula from research:
```python
if bits_per_weight is not None:
    # Approximate rank from target bpw using average hidden dim
    # bpw ≈ rank * (d_out + d_in) / (d_out * d_in)
    # For square matrices: bpw ≈ 2*rank/d, so rank ≈ bpw * d / 2
    # Use first linear layer's dimensions as reference
    first_linear = None
    for m in model.modules():
        if isinstance(m, nn.Linear):
            first_linear = m
            break
    if first_linear is not None:
        d = min(first_linear.weight.shape)
        suggested_rank = max(1, int(bits_per_weight * d / 2))
        print(f"  bits_per_weight={bits_per_weight} -> suggested rank={suggested_rank} (d={d})")
        rank = suggested_rank
```
