<objective>
Add self-contained checkpoint support by collecting shared layers (norms, embeddings) as FP16 into the quantized safetensors file, plus architecture auto-detection.

Purpose: MOE-03 requires checkpoints to be self-contained — usable without the original FP16 model. This means RMSNorm/LayerNorm weights, embed_tokens, and lm_head must be saved as FP16 tensors in the binary factors safetensors file. Architecture detection (via AutoConfig.from_pretrained) is added to quantize.py to log the model type and enable future architecture-specific behavior.

Output: Updated checkpoint.py with collect_shared_layers; updated quantize.py with detect_architecture call and shared layer save.
</objective>

Import nn from torch at module level if not already imported.
